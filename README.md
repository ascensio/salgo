# salgo

This library provides implementations for commonly used algorithms written in Scala.

Sorting-Algorithms
------------------
* BitonicSort
* BubbleSort
* CocktailSort
* CombSort
* CycleSort
* GnomeSort
* HeapSort
* InsertionSort
* MergeSort
* OddEvenSort
* QuickSort
* RadixSort
* SelectionSort
* ShakerSort
* ShellSort
* SlowSort
* StoogeSort
* SwapSort


Cycle-Detection-Algorithms
--------------------------
* Brent-Algorithm
* Floyd-Algorithm


Linear-Algebra-Algorithms
-------------------------
* Gauss-Jordan-Algorithm


Number-Theory
-------------
* Greatest common Divisor (Euclid, GDC)
* Primes (Eratosthenes, Sundaram)


Geometry
--------
* Structures (Point, Vector, Matrix, Plane)
* Closest pair of points
* Gift-Wrapping
* QuickHull
* Graham Scan


Sequence-Algorithms
-------------------
* Sub sequences (Kadane)
* Alignment (Dynamic Time Wrapping)
* Shuffling (FisherYates)
* Matching (Bitap)


Statistics
----------
* Variance
* Covariance
